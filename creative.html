Using CSS to create Art

Following this tutorial, you will see how I added each image to the website to create this magnificant piece of art!

Images found in example

  • clouds gif
  • sun gif
  • tree gif
  • fox gif

NOTE to use the CSS code, insert the name into a div class

For the ground I used the following CSS code

.hero-ground {
background-color: green;
width: 100%;
height: 304px;
}

which should produce a green screen.

Next, we should layer add the layer for the sky

CSS code :

.hero-sky {
background-color: lightskyblue;
width: 100%;
height: 204px;
}

We should get the following background after putting the sky div inside the ground div:

NICE JOB! You just created a nice base for a piece of art.

Now, its time to add the tree and animal gifs

CSS code :

.hero-trees {
background-image: url("images/moving_tree.gif");
background-size: 270px;
width: 100%;
height: 304px;
background-repeat: repeat;
}

.hero-fox {
background-image: url("images/fox_spinning.gif");
background-size: 50px;
background-position-x: 15px;
background-position-y: 200px;
width: 1281px;
height: 304px;
background-repeat: no-repeat;
}

The div class hierarchy ground -> sky -> trees -> fox, you should get something like this!
